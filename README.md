# Fixora AI – Agentic Home Service Marketplace

## Overview

Fixora AI is an AI-powered home service marketplace that connects customers with verified service professionals such as electricians, plumbers, mechanics, technicians, and other skilled workers. The platform leverages Generative AI to simplify service discovery, worker matching, booking management, and customer support through an intelligent conversational experience.

The project is built using the MERN Stack and follows a scalable full-stack architecture with secure authentication, real-time communication, and AI-powered automation.
